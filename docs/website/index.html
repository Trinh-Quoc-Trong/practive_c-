PhysX Engine

Advanced 3D Physics & Rendering Engine in C++

Giới thiệu

PhysX Engine là một engine vật lý 3D hiệu năng cao được phát triển bằng C++17, tích hợp hệ thống render 3D hiện đại dựa trên Vulkan và kiến trúc ECS (Entity Component System) mạnh mẽ. Dự án này được thiết kế để minh họa các khái niệm phức tạp trong phát triển game và mô phỏng, bao gồm vật lý thời gian thực, đổ bóng PBR, quản lý tài nguyên và scripting với Lua.

Mục tiêu của dự án là cung cấp một nền tảng vững chắc để xây dựng các ứng dụng và game 3D, với sự chú trọng vào hiệu suất, khả năng mở rộng và chất lượng mã.

Các tính năng nổi bật

  • 🚀 Vật lý thời gian thực: Mô phỏng rigid body chính xác, phát hiện va chạm nâng cao (GJK/EPA), và hệ thống ràng buộc linh hoạt.
  • 🎨 Render 3D hiện đại: Pipeline dựa trên Vulkan 1.2+, hỗ trợ PBR, deferred shading, đổ bóng mềm (soft shadows) với Cascaded Shadow Maps.
  • ⚙️ Kiến trúc ECS: Quản lý hàng ngàn entity hiệu quả, hệ thống component linh hoạt và thực thi hệ thống song song.
  • 📜 Hệ thống Scripting: Tích hợp Lua 5.4 cho logic game, với API toàn diện để tương tác với engine.
  • 📦 Quản lý Tài nguyên: Tải tài nguyên bất đồng bộ (models, textures, shaders) và quản lý bộ nhớ hiệu quả.
  • ⌨️ Hệ thống Input linh hoạt: Xử lý đầu vào từ bàn phím, chuột và gamepad thông qua một kiến trúc dựa trên sự kiện.
  • 🔊 Hệ thống Âm thanh: Tích hợp phát lại âm thanh 3D cơ bản (ví dụ: với OpenAL hoặc miniaudio).
  • 🔗 Hệ thống Hạt (Particle System): Mô phỏng các vật thể mềm (soft body) hoặc hiệu ứng đặc biệt bằng cách sử dụng các hạt và ràng buộc lò xo.

Cách Build và Chạy

Để build và chạy dự án này, bạn cần có:

  • CMake 3.15+
  • Trình biên dịch C++17 (GCC 10+, Clang 12+, MSVC 2019+)
  • Vulkan SDK 1.2+
  • GLFW 3.3+
  • Lua 5.4
  • Các thư viện bên thứ ba (EnTT, GLM, Sol2, v.v.) sẽ được tải hoặc quản lý qua CMake.

Các bước

# Clone repository
git clone https://github.com/yourusername/PhysX-Engine.git
cd PhysX-Engine

# Tạo thư mục build
mkdir build
cd build

# Cấu hình và build dự án
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release -j 8

# Chạy ứng dụng demo
./bin/PhysX_Demo

Cấu trúc dự án

Dự án được tổ chức theo cấu trúc module, dễ dàng mở rộng và bảo trì:

PhysX-Engine/
├── cmake/              # Các CMake modules tùy chỉnh
├── docs/               # Tài liệu dự án và trang web
│   └── website/
├── include/            # Header files
│   ├── core/           # Core systems (ECS, math, input, assets)
│   ├── physics/        # Physics engine (rigid body, collision, particles)
│   ├── rendering/      # Rendering pipeline (Vulkan, shaders, mesh)
│   └── scripting/      # Lua integration
├── resources/          # Tài nguyên (shaders, textures, models)
├── src/                # Source code implementation
├── test/               # Unit tests
├── third_party/        # Thư viện bên thứ ba
├── .gitignore
├── CMakeLists.txt
├── LICENSE
└── README.md
